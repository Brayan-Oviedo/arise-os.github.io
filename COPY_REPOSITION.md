## 1. HERO SECTION TRANSFORMATION

### CURRENT (Technical Language)
```
"Tu marca crece hoy."
"Prospección automatizada, tiendas premium y escala de ventas para negocios colombianos que quieren crecer en serio."
```

### REPOSITION → EMOTION-DRIVEN OUTCOME

#### **Option A: Freedom + Growth**
```
HERO HEADLINE
"From invisible to unstoppable in 24 hours."

HERO SUBTEXT
"Stop chasing leads. Let our system find them. Build trust with premium design. Close deals with confidence. All $299. No setup fees. No contracts."
```

**Why:** 
- Problem stated (invisible)
- Solution hinted (system finds them)
- Trust marker (premium design)
- Outcome clarity (close deals)
- Pricing transparency (removes friction)

#### **Option B: Authority + Results**
```
HERO HEADLINE
"Your business deserves the tools that close sales."

HERO SUBTEXT
"Get verified leads from Instagram & Google Maps. Present with a website that converts. Follow up with AI-powered CRM. All in one dashboard. 24-hour delivery guaranteed."
```

**Why:**
- Emotional hook (deserves better)
- Business language (close sales)
- Three clear outcomes
- Urgency (24-hour)
- No jargon

#### **Option C: Speed-to-Revenue (Most Aggressive)**
```
HERO HEADLINE
"Your best month ever starts this week."

HERO SUBTEXT
"500+ verified leads. Premium landing page. CRM dashboard. $299. Done."
```

**Why:**
- Extreme outcome focus
- Speed emphasis (this week)
- Numbered outcomes (7 words each)
- Price confidence
- No explanations needed

## 3. PROCESS CARDS (Detailed Copy)

### Card 01: FIND (Current: Prospecta)

**Current:**
> "Escaneamos Instagram y Google Maps en Colombia. Extraemos contactos verificados con email, teléfono y web — 500+ leads calificados listos en 24 horas."

**Reposition:**
```
HEADLINE: Find 500+ Verified Buyers

BODY:
✓ Instagram business profiles + Google Maps listings scanned
✓ Verified emails, phone numbers, website links
✓ Pre-filtered by industry, location, size
✓ Ready to call, email, or message in 24 hours
✓ Colombia-wide coverage (Bogotá, Medellín, Cali, Barranquilla, +20 cities)

OUTCOME: Start closing deals tomorrow — not next month
```

**Why:** 
- Uses action verbs ("Call", "Email", "Message")
- "Buyers" not "Leads"
- "Pre-filtered" = less work on their end
- "Tomorrow" = urgency + attainability

---

### Card 02: PRESENT (Current: Diseña)

**Current:**
> "Construimos tu tienda o landing page con animaciones scroll-frame cinematográficas. Aspecto de sitio de $10K USD. Entregado como HTML listo para subir."

**Reposition:**
```
HEADLINE: Present Like You're Worth $10K

BODY:
✓ Premium landing page or e-commerce store
✓ Scroll-frame animations (high-end feel, fast load)
✓ Mobile-first. Conversion-optimized.
✓ SSL, domain, email included
✓ Delivered as HTML — works with any hosting
✓ Live in 24 hours

OUTCOME: Your prospect's first impression sells for you
```

**Why:**
- "Present Like..." = aspirational
- Removes jargon ("scroll-frame") → explains benefit ("high-end feel")
- Lists what matters (mobile, conversion, included extras)
- "First impression sells" = outcome language

---

### Card 03: CLOSE (Current: Conecta)

**Current:**
> "Usa el dashboard CRM para rastrear leads, enviar el primer mensaje personalizado por WhatsApp o email, y cerrar ventas con propuestas listas."

**Reposition:**
```
HEADLINE: Close With Control

BODY:
✓ CRM dashboard (all leads in one place)
✓ WhatsApp + Email automations
✓ Track deal stage in real-time
✓ AI-generated proposals (ready to send)
✓ Follow-up sequences (so you never forget)
✓ Analytics (see what converts)

OUTCOME: Turn more conversations into closed deals
```

**Why:**
- "Close With Control" = empowerment
- "All leads in one place" = clarity
- "Turn conversations into closed deals" = revenue focus
- Removes "rastrear" (sounds passive) → uses "track" + action verbs

- Higher conversion (reduced friction)
- Better SEO (keyword-aligned body copy)
